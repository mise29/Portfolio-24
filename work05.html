Hirano Miyu Portfolio 2023

Type : Website
Client : 自主制作

就職活動に向けて、私の経歴やこれまでの制作物をまとめたポートフォリオを制作しました。全体のデザインは、明るく鮮やかなカラーで遊び心のあるレイアウトを使用し、読み手に視覚的な魅力を与えるようにしました。また、フォントやグラフィックの選択にもこだわり、私の個性が反映されるように心がけました。このポートフォリオを通じて、私のスキルや経験、明るさや柔軟性なども感じ取ってもらえるようなデザインにしました。

ターゲット

学校のイベントに見に来られる企業様や沢山の方々

目的

より多くの人に目を通してもらうため。自身の強みや制作物を通した思考を伝え、深く知ってもらうため。

情報設計

どんなデザインをするのかやどんなことができるのかを知ってもらうため、Topで印象付けた後、そのままWorksへ。その後、Aboutページでは、読んでいただけるよう、1枚1枚カードにし、自分の雰囲気が伝わる写真と共に配置しました。

デザイン

全体的に「私らしさ」が伝わるデザインを第一に心掛けました。Topで使用したFontを各々の見出しにも使用し、印象付け且つ統一感を持たせました。また、全体的に余白を持たせ、1つ1つのセクションを目立たせるよう心掛けました。

制作期間

企画・ワイヤーフレーム 1週間
デザイン 2週間
コーディング 3週間

使用ツール

Illusttrator / Figma / Visual Studio Code
