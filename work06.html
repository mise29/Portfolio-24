茨城県観光サイト（仮想）

Type : Website
Client : 自主制作

私自身が茨城県出身なこともあり、茨城県の観光サイトを制作しました。 ユーザーが快適に情報を探せるよう、シンプルで直感的なデザインを採用しました。視覚的な要素を多用し、テキストと画像のバランスを取りながら、サイトの見やすさを追求しました。

ターゲット

茨城県を訪れる国内外の観光客や茨城県在住で、地元の観光スポットや特産品について再発見したい人々。週末のお出かけや贈り物を探している人々

目的

観光客や地域住民の方々に正確で詳細な観光情報を提供すること。訪問者が計画を立てやすくすること。

情報設計

訪問者が必要とする情報を細部まで提供するよう心掛けました。具体的な説明を加えることで、茨城県の魅力を深く理解してもらえるよう工夫しました。

デザイン

各セクションのデザインは統一感を持たせつつも、異なるテーマに合わせたデザインの違いを持たせました。これにより、サイト全体の一貫性を保ちながら、個々のセクションの特徴を際立たせました。

制作期間

企画・ワイヤーフレーム 2日
デザイン 2日
コーディング 1週間

使用ツール

Figma / Visual Studio Code
